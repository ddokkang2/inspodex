# Inspodex Product Marketing Context

Last updated: 2026-04-12

## Product Summary

Inspodex is an AI-friendly visual reference workspace for solo creators and freelance designers. It helps users move from vague visual direction to a usable project board, comparison set, and exportable prompt/search pack.

## Primary Audience

- Freelance brand designers
- UI and product designers who make visual directions or concept boards often
- Solo creators who use image generation tools to develop moodboards and key visuals
- One-person studios building fast proposals, landing concepts, and visual packages

## Primary Job To Be Done

Help users turn an unclear visual direction into a workable reference set, search expansion set, and prompt pack within 10 minutes.

## Core Product Loop

1. Explore references across styles, palettes, lighting, poses, and artists
2. Save promising cards into a project board
3. Compare and compress the direction
4. Export search terms, prompts, and summary notes into the next tool
5. Reopen the board in the next project or iteration

## What The Product Is

- A workspace for compressing visual direction
- A bridge between inspiration browsing and AI-assisted execution
- A board-first research and reuse surface for personal creative work

## What The Product Is Not

- A generic inspiration site for everyone
- A pure API product
- A broad social community product
- A team collaboration suite at this stage

## Core Strengths

- Wide reference coverage across multiple visual categories
- Fast browse-to-search workflow
- Strong fit for users who jump between inspiration and generation tools
- Board, compare, and export features create a real working loop instead of a browsing-only experience

## Current Risks

- The category breadth is strong, but can still feel broad before the value is obvious
- The product message is sharper than the current site architecture
- New users can reach the taxonomy before they understand the intended output
- Research artifacts and workspace entry points are not yet clearly separated

## Messaging Direction

Preferred message:

> Inspodex helps AI-based visual workers turn scattered references into a usable visual direction board.

Avoid positioning as:

- a giant reference database
- an all-in-one inspiration platform
- a service for every designer and creator
- an API-first platform

## Site Goals

1. Clarify what outcome the user gets, not only what categories exist
2. Move first-time users into the workspace with less cognitive load
3. Increase board creation, compare usage, export usage, and repeat visits

## Primary UX Principle

Lead with the task and output first, then reveal the taxonomy and controls.

## Priority Signals To Track

- Click-through from homepage into workspace
- First board save rate
- Board reopen rate
- Compare usage rate
- Export usage rate
- Return usage within 7 days
