# Inspodex 포지셔닝 및 MVP 실행안

기준일: 2026-04-11

이 문서는 [`PRODUCT_STRATEGY.md`](/Users/macmini/Vibecoding/inspodex/PRODUCT_STRATEGY.md)를 바탕으로, 실제 제품 메시지, 무료/유료 경계, MVP 범위를 실행 가능한 수준으로 정리한 문서다.

## 1. 한 줄 정의

Inspodex는 AI 기반 비주얼 작업자가 막연한 방향성을 빠르게 레퍼런스 세트와 프롬프트 묶음으로 바꾸는 워크스페이스다.

## 2. 메인 타겟

### 가장 먼저 맞춰야 할 사용자

- 프리랜서 브랜드 디자이너
- UI/프로덕트 시안 작업이 잦은 디자이너
- AI로 무드보드, 키비주얼, 콘셉트 시안을 만드는 1인 크리에이터
- 빠르게 제안서/랜딩/시안 방향을 잡아야 하는 1인 스튜디오

### 이들이 실제로 원하는 것

- 레퍼런스를 빨리 찾는 것
- 서로 어울리는 레퍼런스를 같이 묶는 것
- 다시 찾을 수 있게 저장하는 것
- 바로 검색어와 프롬프트로 넘기는 것

## 4. 무료 / Pro 기능 경계

| 기능 | Free | Pro | 비고 |
| --- | --- | --- | --- |
| 기본 탐색 | O | O | 유입 핵심 |
| 기본 필터 | O | O | 무료 유지 |
| 카드 상세 보기 | O | O | 무료 유지 |
| 외부 검색 연결 | O | O | 핵심 유입 루프 |
| 최근 본 항목 | O | O | 최소 수준 제공 |
| 프로젝트 보드 생성 | - | O | 유료 핵심 |
| 보드별 저장/정리 | - | O | 유료 핵심 |
| 후보 비교 | - | O | 유료 핵심 |
| Prompt / Search Export Pack | 일부 | O | 베타에서는 일부 무료 체험 가능 |
| 조합 추천 묶음 | - | O | 차별화 포인트 |
| 프로젝트 템플릿 | - | O | 재사용 가치 |

### Free의 역할

- 많은 사용자가 부담 없이 들어와 탐색하게 만든다
- 레퍼런스 탐색과 외부 검색 확장의 가치를 즉시 느끼게 만든다
- 하지만 저장과 재사용은 제한해 "작업 공간 가치"를 유료로 남긴다

### Pro의 역할

- 반복 작업 시간을 줄여준다
- 프로젝트 단위로 다시 쓰게 만든다
- 레퍼런스 탐색을 실제 생산 흐름으로 바꿔준다

## 5. 첫 MVP에서 반드시 들어가야 할 것

### MVP 목표

사용자가 실제 프로젝트 단위로 레퍼런스를 저장하고, 다시 열고, 결과물을 밖으로 내보내게 만드는 최소 루프를 검증한다.

### Must Have

- 보드/덱 생성
- 카드 저장
- 저장한 카드 목록 보기
- 보드 단위 이름 수정
- 보드 안에서 간단 비교
- 검색어/프롬프트/팔레트 Export Pack
- 최근 작업 복원

### Nice to Have

- 보드 복제
- 보드 템플릿
- 추천 조합 자동 제안
- 보드 썸네일 자동 생성

### Out of Scope

- 팀 코멘트
- 멀티유저 협업
- 실시간 동기화
- 공개 커뮤니티 피드
- 외부 API 상품화
- 모바일 앱

## 6. 첫 MVP에서 가장 중요한 사용자 루프

추천 핵심 루프:

1. 사용자가 레퍼런스를 탐색한다
2. 마음에 드는 카드를 보드에 저장한다
3. 보드에서 조합을 압축한다
4. Export Pack을 복사한다
5. 다음 프로젝트에서 같은 보드를 다시 연다

이 루프가 돌아야 유료 전환 논리가 생긴다.

## 7. 차별화 포인트를 더 선명하게 만드는 기능 문장

### 현재 문장보다 더 좋은 방향

좋은 레퍼런스가 많다

보다는

작업용 비주얼 방향 세트를 더 빨리 만든다

가 훨씬 강하다.

### 차별화 문장 예시

- 스타일 하나를 찾는 데서 끝나지 않고, 작업에 필요한 레퍼런스 묶음까지 만든다
- 영감 탐색에서 프롬프트 작성까지 한 흐름으로 이어진다
- 검색 사이트를 떠돌지 않고, 바로 다음 작업 단계로 넘어간다

## 8. 가격 테스트 초안

정식 가격을 지금 확정할 필요는 없지만, 테스트 기준은 있어야 한다.

### 추천 테스트 범위

- 월 7,900원 ~ 12,900원
- 연간 할인 플랜 별도 테스트 가능

### 왜 이 구간인가

- 개인 창작자가 부담 없이 시도 가능한 범위
- 단순 탐색 도구보다는 높은 가치를 기대할 수 있는 범위
- 초반 검증 단계에서 심리 장벽이 지나치게 높지 않음

## 9. 앞으로 4주 개발 우선순위

### Week 4

- 실제 사용자 테스트
- 지불 의향 인터뷰
- 무료/Pro 경계 조정

## 11. 지금 메인페이지에서 줄여야 할 것

현재 UI는 기능이 많고 탐색 폭이 넓다는 장점이 있지만, 첫 화면에서 "어디에 집중해야 하는지"가 다소 퍼져 보일 수 있다.

메인페이지에서 줄여야 할 것:

- 첫 화면에 너무 많은 컨트롤 노출
- 디렉터리 설명 없이 기능만 먼저 보이는 구조
- 탐색 자체를 결과물처럼 보이게 만드는 표현

메인페이지에서 강조해야 할 것:

- 어떤 작업 결과를 더 빨리 만드는가
- 어떤 사용자에게 가장 유용한가
- 왜 저장과 조합이 중요한가

## 12. 최종 권장안

지금 Inspodex는 아래 방향으로 가는 것이 가장 합리적이다.

> 무료로는 훌륭한 탐색 허브를 유지하고,
> 유료로는 저장, 조합, 재사용, Export를 제공하는 개인 작업용 비주얼 워크스페이스로 발전시킨다.

즉:

- 카테고리는 넓게 유지한다
- 메시지는 좁게 가져간다
- 결제 포인트는 탐색이 아니라 작업 재사용에 둔다
- API는 지금이 아니라 나중에 다시 본다
