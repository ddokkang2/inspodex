사이트를 고르고, 카드나 입력값을 기준으로 바로 확장 검색합니다.
1 사이트 선택 → 2 카드 클릭 또는 직접 입력 → 3 검색
카드를 먼저 클릭하면 검색어가 자동으로 채워져요. 아니면 직접 입력 후 검색하세요.
