Inspodex는 스타일, 팔레트, 라이팅, 포즈, 작가 레퍼런스를 넓게 탐색하게 해주면서도, 결국 프로젝트 보드와 프롬프트로 이어지는 개인 작업용 워크스페이스로 진화하는 방향을 기준으로 설계하고 있습니다.

탐색

스타일 찾기
이 사이트 안에서 카드 목록을 실시간으로 필터링해요.
1 디렉토리 → 2 검색/필터 → 3 카드 클릭
Esc로도 검색/필터를 초기화할 수 있어요.
/ = 검색 포커스 · Esc = 지우기
키워드로 외부 검색
사이트를 먼저 선택하고, 카드(키워드)를 고른 다음 검색 버튼을 눌러요.
1 사이트 선택 → 2 카드 클릭 또는 직접 입력 → 3 검색
카드를 먼저 클릭하면 검색어가 자동으로 채워져요. 아니면 직접 입력 후 검색하세요.
검색 확장
확장 칩은 1개만 적용돼요.
프롬프트 프리셋
팔레트 카드의 “프롬프트 복사”에 적용돼요.
Enter로도 검색할 수 있어요.
